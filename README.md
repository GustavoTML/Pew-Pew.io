# 🔫 pew-pew.io

pew-pew.io é um jogo top-down shooter inspirado em jogos como Vampire Survivors, desenvolvido como um projeto pessoal durante meus estudos com a Godot.

Foi um dos jogos mais desafiadores que já fiz até aqui, principalmente pela quantidade de sistemas implementados e pela variedade de mecânicas.

## 🎮 Sobre o jogo

O jogador enfrenta hordas de inimigos em uma arena fechada, controlando o personagem com movimentação livre (WASD) e mirando com o mouse.

O objetivo é sobreviver às ondas de inimigos e derrotar os bosses que aparecem ao final de cada horda.

## 🧟 Inimigos

O jogo conta com diferentes tipos de slimes, cada um com comportamentos distintos:

- 🟢 Slime padrão
- 🛡️ Slimes resistentes
- ⚡ Slimes velozes
- ☠️ Slimes venenosos
- 🎯 Slimes atiradores

Ao final de cada horda, o jogador enfrenta um **Boss Slime**, com múltiplos padrões de ataque.

## 🔫 Sistema de armas

O jogador pode evoluir durante a partida e trocar de armas ao subir de nível.

O jogo possui **17 armas**, divididas por raridade:

- ⚪ Common  
- 🔵 Rare  
- 🟣 Epic  
- 🟡 Legendary  

Entre as armas disponíveis:

- Pistolas
- Rifles
- Metralhadoras
- Lança-chamas
- Minigun
- Sniper teleguiada (sim, isso mesmo)

## 🧠 Desafio

O jogo possui dificuldade progressiva, com inimigos cada vez mais fortes e hordas mais intensas.

Você consegue derrotar o último boss no modo hardcore?

## 🎨 Observações

Este projeto foi desenvolvido com foco em aprendizado e experimentação, então o código pode conter soluções não otimizadas ou abordagens mais simples.

Ainda assim, foi um dos projetos que mais gostei de desenvolver até agora.
